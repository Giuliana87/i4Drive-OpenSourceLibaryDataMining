File for analysis of SHRP2 dataset. Request the dataset to https://insight.shrp2nds.us

1- script_MCA_data_event.R: code for data event manipulation and for Multivariate Correspondance Analysis (MCA).
2- script_processing_data_event_driver.R: code for data event and data driver manipulation
3- script_trees_event.R: code for Classification Tree on event data, analysis of contingency tables in terminal nodes, and Random Forest on event data.
